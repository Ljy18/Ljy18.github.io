减肥与生存

发表于 2023-08-20 分类于 Daily 本文字数： 858 阅读时长 ≈ 1 分钟

减肥

本质上减肥这个东西是从二十世纪以来的新兴事物,在二十世纪以前肥胖并不认为是一个问题,它被认为是富裕的象征,而不是现在肥胖已经成为贫穷的标志,如果它跟一些有钱的联系起来那就是暴发户,土豪,这些明显带有贬义的词汇联系在一起.这个本质上其实是和人类的生产力在工业革命以后有一个迅猛的大发展有很大的关系,就是普通人也可以获得足够的热量摄入所以导致的肥胖,当普通人也越来越胖的时候,其实有富人主导,或者是说由上层阶级主导的审美观念即开始发生了转变,我们都知道在以前,人们其实是以胖为美,或者说是以丰盈为美,比如以前的美国的巨星玛丽莲梦露,其实她就不是瘦的,就是丰盈. 所以当普通人也越来越胖的时候,瘦逐渐开始成为一个新兴的审美方向,其实这件事情很简单,说白了就是稀有化,这个是审美标准转变的一个决定性因素,特别是由于上层开始转变,然后中产阶级开始跟上,然后逐渐的审美标准就自然而然就变了,这时候你可能会问,那穷人呢?很遗憾就是这里其实没有什么穷人的事情,因为穷人最先放弃就是形象,说到底就是审美就是由上层决定,然后中产跟进.然后由于中产对于社会舆论空间的占据,然后形成了一个当代的一个审美标准.其实这种审美标准逐渐开始形成一个歧视,就是肥胖歧视,当然一个公司招人的时候,他们会优先找那些瘦的人.他们会认为这些肥胖是一种对自己不负责任,也没有办法很好的完成公司的任务,从而招聘的时候优先把胖的人剔除.

生存

其实吧,从历史上看我们发现在农业文明之前,我们的身材体重应该是最好的,因为我们会摄入大量的蛋白质,还有就是优胜劣汰我们需要的打猎,瘦小的身材其实根本无法完成任务,所以他们都死了.但是这情况到了农业文明开始发生了转变.由于的耕种其实并不需要身材硕大威猛,所以很瘦小的也能干,还有就是在农业文明世代是没有节育的,所以就是能生多少就生多少,所以导致绝大部分的人处于吃不饱也饿不死的状态,这样就会让为此一个刚好的可以下地干农活的一个体型,也就会极其之瘦.这个也是为什么古时候以胖为美的真正原因,因为稀有,能吃胖的都是贵族和有钱人.

减肥记录

发表于 2023-08-14 更新于 2023-08-18 分类于 Daily 本文字数： 416 阅读时长 ≈ 1 分钟

起因

第一个就是我好像从小就比较胖,但是也不是特别胖,就是从小有小肚子,初中以后由于住校吃饭也不是特别注意所以导致我的体重一直在上升,到了高中就已经有160多斤了,到了疫情期间由于不能出门导致我的体重更是来到恐怖的180多斤,这个体重直接让我第一次痛风,我也是第一次意识到胖所能带来的危害,然后我吃医生开的一些减脂药将体重控制在170左右然后维持两年,然后我通过减少饭量将体重控制在160左右,但是发现还是要瘦一点好,反正现在有时间所以我又开始减肥了.

方法和结果

一开始就是首先开始少吃一顿,就是只吃早餐和中午饭晚饭就不吃,然后每天晚上沿着校园走两圈每圈大概是三公里,一圈大概要40分钟吧!使用这个节奏发现自己可以一天瘦一斤其实,然后大概是瘦到78kg的时候发现自己的体重又开始没有办法下降,然后我开始每天早上只吃两个鸡蛋,然后隔两天吃一顿正餐,然后瘦到现在的76.4kg,其实我最终的目标为65kg,然后控制在这个体重上,继续加油吧!

关于献血的一些思考

发表于 2023-06-16 分类于 news 本文字数： 441 阅读时长 ≈ 1 分钟

起因

就是我一直从网上看,特别是知乎等平台,国内的献血一直因为以前的一些丑闻,比如上世纪的九十年代的买血事件,还有近十年的如2009年的用血浇花事件,这些事件一直导致国内的献血制度和一些操作一直为人诟病

搜寻

不过我去搜寻的一些资料如近五年的献血的统计发现,从总数上来说,献血的人次总数是不断上升的,从2014年的1299万上升到2021的2855.9万是之前的两倍,但是从千人的献血率来看的话,从2014的9.5/千人口上升到2021的12就是增长了百分之20,其实增长的并不快,跟欧洲等国家的平均每千人的30相比仍有较大的差距,不过这个和国家的发展水相关,根据WHO的数据全球每年有1.125亿人献血，其中高收入国家人口占大约50%，占世界人口的19％。世界卫生组织数据显示，高收入国家的献血率为每千人32.6次，中上收入国家为15.1次，中低收入国家为8.1次，低收入国家为4.4次.所以不能说是因为一些丑闻什么相关导致中国鲜血率低,从数据上来看,更多可能是因为经济的发展水平问题.
